Instructions

Find the cpu.js file in this repository, and implement all the instructions that are left out.

There exist practice ROMs that only use a limited number of instructions, which means that you can run them in the runner once you have implemented the instructions necessary for that ROM. For your convenience we have added debugger statements in left-to-implement parts, meaning that executing a ROM will automatically break on something you need to do still. (Just open your developer tools)

This repository also includes a set of tests for (and not limited to) all instructions.
You can verify your implementation for an instruction is correct by making the relevant tests pass.

It is recommended that you visit the documentation for CHIP-8, where you can learn what an instruction's execution should do once it is implemented.
